# Billing System Django App

This is a simple Django-based billing system that allows you to manage products, create bills, and view billing history.

## Features
- Manage products (add, update, delete)
- Create bills for customers
- Store product and billing data in PostgreSQL
- Optional REST API support

## Requirements
- Python 3.10+
- PostgreSQL 12+
- Django 4.2+
